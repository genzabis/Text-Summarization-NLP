Tentang Project

Aplikasi ini mengimplementasikan tiga pendekatan text summarization untuk Bahasa Indonesia di atas dataset IndoSum:

  • TextRank – baseline ekstraktif berbasis graf kalimat dengan skor TF-IDF + cosine + PageRank.
  • Seq2Seq – encoder-decoder GRU bidirectional dengan Bahdanau attention (PyTorch). Dilatih pada split train IndoSum, di-evaluasi via ROUGE.
  • Google Gemini API – peringkasan abstraktif via model gemini-1.5-flash menggunakan API key dari file .env.

Cara Menjalankan

  1. Install dependency: pip install -r app/requirements.txt
  2. Salin app/.env.example menjadi app/.env, isi GEMINI_API_KEY.
  3. (Opsional) Latih Seq2Seq: cd app && python -m app.train --epochs 5 --max-train 20000
  4. Jalankan server: python -m app.backend.app
  5. Buka http://127.0.0.1:5000 di browser.
