Tentang Project

Aplikasi ini mengimplementasikan dua pendekatan text summarization untuk Bahasa Indonesia di atas dataset IndoSum:

  • TextRank — baseline ekstraktif berbasis graf kalimat dengan skor TF-IDF + cosine + PageRank.
  • Claude 4.5 Sonnet API — peringkasan abstraktif menggunakan API key dari file .env.

Cara Menjalankan

  1. Install dependency: pip install -r app/requirements.txt
  2. Salin app/.env.example menjadi app/.env, isi GEMINI_API_KEY.
  3. Jalankan server: python -m app.backend.app
  4. Buka http://127.0.0.1:5000 di browser.
