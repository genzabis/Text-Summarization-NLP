Abstrak

Proyek ini membandingkan tiga pendekatan peringkasan teks Bahasa Indonesia di atas dataset IndoSum: TextRank sebagai baseline ekstraktif berbasis graf, Seq2Seq + Bahdanau attention yang dilatih dari nol di PyTorch, dan Google Gemini 1.5 Flash sebagai pembanding LLM. Sistem dijalankan end-to-end melalui Flask REST API dengan UI HTML/CSS/JS murni, mencakup pipeline preprocessing, training, evaluasi ROUGE, hingga visualisasi alignment attention.

01

Latar Belakang & Tujuan

Volume teks digital Bahasa Indonesia (berita, artikel, dokumen resmi) tumbuh jauh lebih cepat dari kemampuan manusia membacanya. Text summarization menyaring inti dokumen panjang menjadi ringkas tanpa kehilangan informasi penting, sekaligus mengurangi bias & halusinasi pada sistem AI generatif.

Tujuan Proyek

  • Membangun model peringkas Seq2Seq + Bahdanau attention untuk Bahasa Indonesia.
  • Menyediakan baseline TextRank sebagai pembanding kuantitatif (ROUGE).
  • Mengintegrasikan Google Gemini API sebagai pembanding LLM abstraktif.
  • Menyediakan antarmuka web ringan untuk demo & evaluasi end-to-end.
02

Metode & Pendekatan

EXT

TextRank · Baseline

Graph-based: TF-IDF + cosine similarity → PageRank iteratif. Memilih kalimat asli dengan skor tertinggi (ekstraktif).

ABS

Seq2Seq + Bahdanau Attention · Model Utama

Encoder GRU bidirectional → additive attention dinamis → decoder GRU + softmax. Dilatih dari scratch di IndoSum (PyTorch).

LLM

Google Gemini 1.5 Flash · Pembanding

API abstraktif via google-generativeai. Prompt instruksi Bahasa Indonesia, jumlah kalimat dapat diatur.

Dataset & Preprocessing

IndoSum (Kurniawan & Louvan, 2018) — dataset peringkasan berita Bahasa Indonesia berformat .jsonl, berisi pasangan artikel & gold summary buatan manusia.

14k+ Train

Pipeline Preprocessing

  1. Flatten struktur paragraph → sentence → token
  2. Normalisasi tanda baca & lowercase
  3. Tokenisasi regex Bahasa Indonesia
  4. Build vocab + special token <pad><bos><eos><unk>
  5. Pad & truncate max-src=120 · max-tgt=25 token

Kesimpulan

  • Sistem peringkas teks Bahasa Indonesia berhasil dibangun end-to-end: model + backend Flask + frontend HTML/CSS/JS.
  • TextRank tetap juara praktis di lingkungan tanpa GPU dengan ROUGE-1 ≈ 0.43 — konsisten dengan literatur ekstraktif Bahasa Indonesia.
  • Pipeline Seq2Seq + Bahdanau attention sudah konvergen (val_loss 6.94 → 6.63), tetapi butuh GPU + dataset penuh untuk kompetitif.
  • Integrasi Gemini API menyediakan opsi LLM abstraktif untuk pembanding kualitatif.
  • Kontribusi: arsitektur modular, dokumentasi training/benchmark, dan UI tema light yang siap demo pameran.

Saran Pengembangan

Latih ulang Seq2Seq di GPU/Colab dengan ≥30 k sampel × ≥10 epoch (emb=128, hidden=256), tambah beam search, dan eksplorasi transformer pre-trained Bahasa Indonesia (mBART, IndoBART, IndoT5).
